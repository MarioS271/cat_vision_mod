# CatVision

A fabricmc minecraft mod which gives you Night Vision, even if you aren't a server operator. Inspired by CatEyes by Tee6! :)

With this mod, you can give yourself night vision with the click of a button.
The mod also has options to automatically enable night vision.
And the best part: you don't need any permission, just use it anywhere!

**DISCLAIMER: I AM NOT LIABLE IF YOU GET BANNED FOR CHEATING USING THIS, PLEASE FOLLOW THE RULES OF WHATEVER
SERVER/REALM/MULTIPLAYER WORLD YOU ARE PLAYING ON WHEN USING THIS MOD!**

### Current Features (v1.4):
- Smart Client Night Vision
- Darkness/Blindness/Nausea Immunity
