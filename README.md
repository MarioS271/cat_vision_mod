# cat-vision-mod (fabric)
A minecraft mod which gives you Night Vision, even if you aren't a server operator. Inspired by CatEyes by Tee6! :)

### Current Features (v1.1):
- (Smart) Client Night Vision!

### Languages:
- English
- German
- German (Austrian)
